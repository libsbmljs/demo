# libsbmljs demo

## Intro

This is the source code for the libsbmljs demo

## Installing dependencies

```
npm install
```

## Building

Development:

```
DATABASE_PREFIX=/path/to/database npm run start
```

Production:

```
DATABASE_PREFIX=x npm run build
```
